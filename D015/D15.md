## Question 1 ( In Github )  
**Problem:**  
Given a matrix, calculate the sum of its primary diagonal elements. The primary diagonal consists of elements where row index equals column index.

**Code:**  
```c
#include <stdio.h>
int main(){
    int r;
    int c;
    scanf("%d %d",&r,&c);
    

    int m[r][c];



    for(int i=0;i<r;i++){
        for (int j = 0; j < c; j++) {
            scanf("%d", &m[i][j]);
        }
    }
    int sum=0;
    if(r<=c){
        for(int i=0;i<r;i++){
            sum+=m[i][i];
        }

    }
    else{
        for(int i=0;i<c;i++){
            sum+=m[i][i];
        }
    }
    printf("%d",sum);
    return 0;
}   
```

**Sample Output:**  
```
3 3
1 2 3
4 5 6
7 8 9
15
```
