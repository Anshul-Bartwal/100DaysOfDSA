## Question 1 ( In Github )  
**Problem:**  
Implement linear search to find key k in an array. Count and display the number of comparisons performed.

**Code:**  
```c
#include <stdio.h>
int main(){
    int n;
    printf("Enter n: ");
    scanf("%d",&n);
    int arr[n];
    printf("Enter Array elements: ");
    for(int i=0;i<n;i++){
        scanf("%d",&arr[i]);
    }
    printf("Enter value to find: ");
    int key;
    scanf("%d",&key);
    for( int i=0;i<n;i++){
        if(arr[i] == key){
            printf("%d",i);
            return 0;
        }
    }
    printf("Not found");
    return 0;
    

}
```

**Sample Output:**  
```
Enter n: 5
Enter Array elements: 10 20 30 40 50
Enter value to find: 20
1
```
