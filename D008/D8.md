# Day 08 - DSA Problem Solutions

## Question 1 ( In Github )  
**Problem:**  
Given integers a and b, compute a^b using recursion without using pow() function.

**Concept Used**
- 

**Code:**  
```c
#include <stdio.h>
int power(int a,int b){
    if(b==0){
        return 1;
    }
    return a*power(a,b-1);

}

int main(){  
    int a,b;
    scanf("%d %d",&a,&b);

    printf("%d",power(a,b));
    return 0;

}
```

**Sample Output:**  
```
2 5
32
```
