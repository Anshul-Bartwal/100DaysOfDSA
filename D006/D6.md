## Question 1 ( In Github )  
**Problem:**  
Given a sorted array of n integers, remove duplicates in-place. Print only unique elements in order.

**Code:**  
```c
#include <stdio.h>
int main(){
    int n;

    
    scanf("%d", &n);

    int arr[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }
    int ans[n];
    int ansIndex=0;
    ans[0]=arr[0];
    for(int i=1;i<n;i++){
        if(arr[i] == ans[ansIndex]){
            continue;
        }
        ans[++ansIndex]=arr[i];


    }

    for (int i = 0; i < ansIndex+1; i++) {
        printf("%d ", ans[i]);
 
    }
    return 0;
}
```

**Sample Output:**  
```
8
1 1 1 2 2 3 4 4
1 2 3 4 
```
