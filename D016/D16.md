# Day 16 - DSA Problem Solutions

## Question 1 ( In Github )  
**Problem:**  
Given an array of integers, count the frequency of each distinct element and print the result.

**Code:**  
```c
#include <stdio.h>

int main() {
    int n;
    int arr[100];
    int freq[1001] = {0}; //assumption range of values till 1001 
    
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
        freq[arr[i]]++;
    }

    for (int i = 0; i < n; i++) {
        if (freq[arr[i]] > 0) {
            printf("%d:%d ", arr[i], freq[arr[i]]);
            freq[arr[i]] = 0;
        }
    }
    printf("\n");
    
    return 0;
}
```

**Sample Output:**  
```
5
1 2 2 0 0
1:1 2:2 0:2 
```
