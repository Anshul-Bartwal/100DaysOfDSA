## Question 1 ( In Github )  
**Problem:**  
Given an array of integers, count the number of subarrays whose sum is equal to zero.

**Concept Used**
- 

**Code:**  
```c
#include <stdio.h>

int main() {
    int n;

    scanf("%d", &n);
    int arr[n];
    
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }
    
    int count = 0;
    
    // Brute force
    for (int i = 0; i < n; i++) {
        int sum = 0;
        for (int j=i;j<n;j++) {
            sum +=arr[j];
            if (sum==0) {
                count++;
            }
        }
    }
    
    printf("%d\n", count);
    return 0;
}
```

**Sample Output:**  
```
4
1 -1 2 -2
3
```
