# Day 09 - DSA Problem Solutions

## Question 1 ( In Github )  
**Problem:**  
A secret system stores code names in forward order. To display them in mirror format, you must transform the given code name so that its characters appear in the opposite order.

**Code:**  
```c
#include <stdio.h>
void rev(char *str){
    if(*str == '\0'){
        return; 
    }
    rev(str+1);
    printf("%c",*str);

}
int main(){
    char str[100];
    scanf("%s",str);
    char *p = str;
    rev(p);
    return 0;

}
```

**Sample Output:**  
```
hello
olleh
```
