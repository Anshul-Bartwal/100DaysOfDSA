# Day 7 - DSA Problem Solutions

## Question 1 ( In Github )  
**Problem:**  
Write a recursive function fib(n) to compute the n-th Fibonacci number where fib(0)=0 and fib(1)=1.

**Concept Used**
- Memoization

**Code:**  
```c
#include <stdio.h>
int memo[1000] = {0};
int fib(int n){
    if (n==0){
        return n;
    }
    if(n==1)return 1;

    if (memo[n] !=0){
        return memo[n];
    }
    
    memo[n]= (fib(n-1)+fib(n-2));
    return memo[n];
}

int main(){
    int n;
    scanf("%d",&n);
    printf("%d",fib(n));
    return 0;
}
```

**Sample Output:**  
```
15
610
```
